# AI Note Bot

Telegram-бот для управления заметками с использованием локальных ИИ моделей.

## Возможности
- **Добавление заметок**: Отправляйте текстовые или голосовые сообщения. Бот автоматически транскрибирует (если это голос), суммирует и тегирует их.
- **Поиск**: Используйте `/search <запрос>` для семантического поиска заметок.
- **Анализ**: Используйте `/analyze` для получения инсайтов и планов на основе всех ваших заметок.

## Установка

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

> **Примечание**: Первый запуск автоматически скачает модели (~2-3GB). Это может занять несколько минут.

### 2. Переменные окружения

Создайте файл `.env` в корне проекта:
```env
TELEGRAM_TOKEN=ваш_токен_бота
LLM_MODEL_NAME=microsoft/phi-2
WHISPER_MODEL_SIZE=base
```

- Получите `TELEGRAM_TOKEN` у [@BotFather](https://t.me/BotFather).
- `LLM_MODEL_NAME`: модель для текстовых задач (по умолчанию `microsoft/phi-2`, ~2.7GB).
- `WHISPER_MODEL_SIZE`: размер модели Whisper (`tiny`, `base`, `small`, `medium`, `large`).

### 3. Запуск бота
```bash
python main.py
```


## Использование
- Запустите бота командой `/start`.
- Отправьте любой текст, чтобы сохранить его как заметку.
- Отправьте голосовое сообщение, чтобы транскрибировать и сохранить его.
- Поиск: `/search список продуктов`
- Анализ: `/analyze`

## Системные требования
- **RAM**: минимум 4GB (рекомендуется 8GB+)
- **CPU**: многоядерный процессор (GPU опционально для ускорения)
- **Диск**: ~3-5GB для моделей
